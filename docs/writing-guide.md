# 晄希の書き方ガイド

iPad の Safari で GitHub を開いて書くよ。

## はじめかた

1. https://github.com/nkhrhr/MineCraft/issues を開く
2. 「New issue」をタップ
3. 「物語のアイデア」を選ぶ
4. テンプレートを埋めて「Submit」

## Markdown の書きかた（少しずつ覚えよう）

### レベル1：そのまま書く

```
きょう思いついたこと。
森の奥にドラゴンがいる。
```

### レベル2：箇条書き

```
- 森の奥にドラゴンがいる
- ドラゴンは実はやさしい
- 話しかけると仲間になる
```

こうなる：
- 森の奥にドラゴンがいる
- ドラゴンは実はやさしい
- 話しかけると仲間になる

### レベル3：見出しをつける

```
## キャラクター

**ドラゴンのリュウ** はやさしいドラゴン。

## ストーリー

1. 森でリュウに出会う
2. リュウが道を教えてくれる
3. いっしょにボスを倒す
```

### レベル4：もっとくわしく

```
## 第四章：ドラゴンの谷

> 「よく来たな、小さき勇者よ」

ドラゴンの **リュウ** が目の前に降り立った。
リュウは見た目はこわいけど、実は村を守っていた。

### もらえるアイテム
- ドラゴンの鱗（ぼうぎょ力アップ）
- 炎の剣（攻撃力30）

### コマンド
`/give @p diamond_sword`
```

## プレビューのしかた
